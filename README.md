# CS23333-Object-Oriented-Programming-Using-Java
Covers: Key aspects Of Oops and Java
A hands-on lab record that teaches and demonstrates Object Oriented Programming principles in Java through incremental experiments and practical exercises. Each lab focuses on a single concept, contains fully working example code, unit tests, expected outputs, and short reflective questions for learners.
Key Topics Covered
• 	Classes and Objects
• 	Encapsulation and Access Modifiers
• 	Constructors and Constructor Overloading
• 	Inheritance and Method Overriding
• 	Polymorphism and Dynamic Binding
• 	Abstract Classes and Interfaces
• 	Packages and Modularization
• 	Exception Handling and Custom Exceptions
• 	Collections and Iteration Patterns
• 	File I/O and Serialization
